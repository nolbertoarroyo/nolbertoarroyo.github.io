Plans Nearby

Plans Nearby helps users find things to do in their area by displaying events and popular places to go using their current location or city input. App uses Foursquare API, Facebook SDK, and Eventbrite API. Available in the Play Store.

  • Project Page: Github
  • Date: September 2016
  • General Assembly: Project 4

Vice 2.0

Vice 2.0 uses Vice News API to display current articles from various categories. Users can save articles they like to read offline, and can share articles on facebook. This is a group project for the General Assembly immersive program. This application was completed in a week long sprint in a group of 3. Project available on GitHub page.

  • Project Page: Github
  • Date: August 2016
  • General Assembly: Project 3

My J's

My J's is my second project in the General Assembly immersive program. Our task was to make a commerce app and use the skills learned in the first month of class. I used a viewPager, SQLite database, intents, and fragments to complete the project. Users can search for shoes by year or color. Code available on GitHub.

  • Project Page: Github
  • Date: June 2016
  • General Assembly: Project 2
